🔗 Connect Google Sheets (Live Data)
Uses JSONP — works when opening the HTML directly from your computer. No web server needed.
1
Open your Google Sheet → Extensions → Apps Script. Delete any existing code and paste this:
function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = sheet.getDataRange().getValues(); var headers = data[0]; var rows = data.slice(1).map(function(row) { var obj = {}; headers.forEach(function(h, i) { obj[h] = row[i]; }); return obj; }); var cb = e && e.parameter && e.parameter.callback ? e.parameter.callback : null; if (cb) { // JSONP mode return ContentService.createTextOutput(cb + '(' + JSON.stringify(rows) + ')') .setMimeType(ContentService.MimeType.JAVASCRIPT); } // JSON mode (fetch with CORS) return ContentService.createTextOutput(JSON.stringify(rows)) .setMimeType(ContentService.MimeType.JSON); }
2
Click Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Click Deploy. Copy the Web app URL.
3
Paste the URL below. It should look like:
https://script.google.com/macros/s/ABC123.../exec
